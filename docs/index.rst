=====================
Library Scarlet Devil
=====================

Добро пожаловать в **Library Scarlet Devil**.

Это библиотека статей, заметок и гайдов, собранная на базе Sphinx и Read the Docs.
Здесь можно хранить технические материалы, игровые заметки, лорные записи, проектные документы и любые другие тексты.

.. note::

   Библиотека только получила основу. Дальше её можно расширять разделами, категориями и отдельными статьями.
