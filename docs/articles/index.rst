======
Статьи
======

Этот раздел предназначен для отдельных статей, гайдов и заметок.

Как пользоваться разделом
=========================

1. Создай новый файл в папке ``docs/articles/``.
2. Возьми за основу ``template.rst``.
3. Добавь новую статью в ``toctree`` ниже.
4. Сохрани изменения и дождись сборки Read the Docs.
